## git 명령어 요약

- clone : 원격 저장소 복사
- add : 스테이지 영역에 작업 파일 추가
- commit : 세이브, 스테이지 영역의 파일들을 가지고 커밋(=세이브)을 만들 수 있다.
- push : 커밋한 파일을 원격저장소에 복사

## 브랜치 변경하기

- 브랜치란 : 기존 내용을 유지한 채 새로운 내용을 추가하고 싶을 때 사용한다.
- 체크아웃 : 특정 브랜치(혹은 커밋) 으로 돌아가고 싶을 때 사용.
- 소스트리의 체크아웃 : 브랜치 이름을 더블클릭하는 것만으로 체크아웃 가능
